# Windows and Linux Scripts
Focus on scripts to generate attacks based on the OS used.  
- - -

## Types
1. Worms:
  * Scripts for an infected machine to spread the worm in the network
2. Wiper Scripts:
  * Caution to prevent the spread of wiper to other machines
3. Enumeration:
  * Scripts to simulate threat attacker enumeration of network, device, server etc
4. Pivoting:
  * Scripts to pivot/privilege escalation.
5. More to be Added
