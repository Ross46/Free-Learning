# Virtual Machine Setup
This topic focuses on Setting up Command and Control Server
- - -

This consists of following features for Windows and Linux OS
  1. Install scripts.
  2. Command and Control server  
    * Manage all VMs.  
    * Manual Control of User.  
    * Customizable by user.  
    * Automate Scenario setup.  
  3. CLI based C2.
  4. Integrate GUI web based C2 (Planned).

- - -

## Note
At present the Install scripts are written for debian buster.  
This topic will be updated regularly.
